# 一、一般基础知识

## 1.数量遗传学基础

|          | 亲子间          | 半同胞间         | 全同胞间                           |
| -------- | --------------- | ---------------- | ---------------------------------- |
| 协方差   | $0.5\sigma_A^2$ | $0.25\sigma_A^2$ | $0.5\sigma_A^2 + \sigma^2(EI, FS)$ |
| 相关系数 | $0.5h^2$        | $0.25h^2$        | $0.5h^2 + C^2$                     |

用亲属间协方差与相关系数估计遗传参数的一个重要前提，亲属间不存在环境协方差组分。
